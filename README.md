# firstwebproject

Статическая демо-страница: форма с полями ввода на **HTML5**, стили **Bootstrap 3**, скрипты **jQuery** и плагин **Masked Input** для маски российского номера телефона.

## Возможности

- Дата, многострочный текст, телефон с маской `8(999) 999-9999`, email с подсказками из `<datalist>`, неотрицательное целое число, загрузка файла.
- Встроенная проверка полей браузером (HTML5); после успешной проверки отправка перехватывается скриптом и показывается сообщение (данные на сервер не уходят — форма учебная).

## Запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
python3 -m http.server 8080
```

Затем откройте в браузере: `http://localhost:8080/`

## Структура

| Путь | Назначение |
|------|------------|
| `index.html` | Разметка формы и подключение стилей/скриптов |
| `css/bootstrap*.css` | Стили Bootstrap |
| `js/jquery-3.2.1.js` | jQuery |
| `js/jquery.maskedinput*.js` | Маска ввода для телефона |
| `js/bootstrap.min.js` | JS-компоненты Bootstrap (на странице почти не используются) |
| `js/main.js` | Инициализация маски и обработка отправки формы |

## Зависимости

Внешних пакетов и сборки нет — только локальные файлы в репозитории.

## Замечания по развитию

- Для реальной отправки данных задайте `action` и `method` у формы и уберите `preventDefault` в `js/main.js`, либо отправляйте через `fetch`.
- При необходимости обновите jQuery и Bootstrap и проверьте совместимость плагина Masked Input.
